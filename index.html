PET SHELTER
APPLICATION

In this project in collaboration with Amazon Cloud Institute, I developed and deployed a fully functional pet shelter application using React and Node.js for the front end and AWS SAM for the backend. The application allows users to browse available pets, view their details, and fill out adoption forms.
The backend is built using AWS SAM, which provides a serverless architecture for handling API requests and managing data storage. I implemented RESTful APIs to handle pet information, user interactions, and adoption requests. The application also integrates with a database to store pet details and user information securely.
